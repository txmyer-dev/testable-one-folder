# Onboarding Checklist (The First 7 Days)

Goal: Move the client from "Chaos" to "Managed" within one week to justify the $750/week rate.

## Phase 1: The "Handshake" (Day 0)
- [ ] **Agreement & First Payment:** Ensure the first $750 is processed via Syncro/Stripe.
- [ ] **Welcome Email:** Send a professional "What to Expect" guide.
- [ ] **Access Collection:** Securely gather admin credentials for the server, firewall, and domain registrar.
- [ ] **Emergency Contact:** Establish the "Bat-Signal" (Slack channel or dedicated phone number).

## Phase 2: The "Infection" (Day 1-2)
*Note: We call this "Infection" because we are spreading our RMM agents across the fleet.*
- [ ] **RMM Deployment:** Install Syncro/Ninja agents on all workstations and servers.
- [ ] **Inventory Sync:** Verify the device count matches the agreement.
- [ ] **Initial Scan:** Run a "Silent Audit" to identify critical security gaps (Missing patches, no AV, failing drives).

## Phase 3: The "Quick Wins" (Day 3-5)
- [ ] **Patching Storm:** Remotely push all critical Windows and 3rd-party updates after hours.
- [ ] **Security Baseline:** Deploy your standard Antivirus/EDR.
- [ ] **Documentation:** Map the network. Document IPs, serial numbers, and ISP account info in Hudu.

## Phase 4: The "First Friday" (Day 7)
- [ ] **Onboarding Report:** Deliver a "State of the Union" report showing:
    *   Total threats blocked.
    *   Total updates applied.
    *   List of devices that need replacement (The "Red List").
- [ ] **Review Call:** 15-minute sync to confirm they feel supported and "seen."
