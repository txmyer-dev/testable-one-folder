# Fractional IT Framework (V1)

This repository contains the documentation and live landing page for the **Fractional IT Department** service framework. 

This model replaces a $60,000/year internal IT hire with a $38,400/year professional managed service department, targeting 25-50 user businesses (Professional Services preferred).

## 🚀 Live Website
The pitch and offerings have been automatically compiled into a high-converting landing page hosted on GitHub Pages:
**[View the Live Site](https://txmyer-dev.github.io/testable-one-folder/)**

## 📂 Repository Structure

### The Pitch & Strategy
* `SESSION_SUMMARY.md` - The core decisions, pricing, and project status.
* `context.md` - The overall sales philosophy, hook, and objection handling.
* `offerings.md` - Pricing structure, what's included, and "Profit Multiplier" add-ons.
* `service-stack.md` - The technical engine (Syncro, Huntress, Bitdefender, etc.).
* `onboarding-checklist.md` - Process for the first 7 days of a new client.
* `references.md` - Reference materials and links.
* `agents.md` - AI agent configuration or notes for this stack.

### The Website Front-End
* `index.html` - The semantic HTML5 landing page.
* `style.css` - Custom, framework-free CSS (dark mode, glassmorphism).
* `script.js` - Micro-interactions and scroll observers.
